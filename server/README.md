## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Start with Prisma

```bash
$ npx prisma init
$ npx prisma generate
$ npx prisma migrate dev --name initial
```
